## BurgerCities Protocol
### Install
```sh
yarn
```
### Cmd
compile
```sh
yarn compile
```
test
```sh
yarn test
```
deploy

```sh
# scripts/deploy.ts
yarn deploy
```
verify
```sh
# scripts/verify.ts
yarn verify
```
clean cache
```sh
npx hardhat clean
```
